## Voice Assistant

The portfolio supports OmniDimension's Web Bot Widget. In the OmniDimension dashboard, open your agent, choose Deploy > Web Bot Widget, and copy the generated script.

Set this environment variable in Vercel or `.env`:

```bash
NEXT_PUBLIC_OMNIDIM_WIDGET_EMBED_SCRIPT="<script ...></script>"
```

If your OmniDimension snippet gives you a script URL and separate values, use these instead:

```bash
NEXT_PUBLIC_OMNIDIM_WIDGET_SCRIPT_URL=
NEXT_PUBLIC_OMNIDIM_WIDGET_AGENT_ID=
NEXT_PUBLIC_OMNIDIM_WIDGET_SECRET_KEY=
```

Do not use `OMNIDIM_API_KEY` in browser-facing variables. The widget secret generated by OmniDimension is separate from the private API key.

For API-key based voice calls, set these server-side variables:

```bash
OMNIDIM_API_KEY=
OMNIDIM_AGENT_ID=
OMNIDIM_FROM_NUMBER_ID=
OMNIDIM_BASE_URL=https://backend.omnidim.io/api/v1
```

Visitors can then enter an E.164 phone number, such as `+919876543210`, in Nami's voice call field. The app calls `/api/voice/call`, which dispatches the call through OmniDimension without exposing your private API key.
